# Efficacy

A guided learning platform designed for self-taught learners to manage learning paths, mentor sessions, tasks, and long-term skill growth.

---

## Why This Project Exists

Most self-taught learners struggle with:

* lack of structure
* inconsistent accountability
* scattered resources
* poor progress tracking

Efficacy was built to solve this problem by combining learning management, mentorship guidance, and productivity tools into one platform.

The goal is not just content consumption, but helping learners build consistent learning systems.
